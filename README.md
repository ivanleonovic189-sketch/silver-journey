# Enter Pay

Платёжная система с мерчантами для high-risk бизнеса (казино, азартные игры).

## Концепция

**Мерчанты** — это посредники, которые:
- Принимают платежи от пользователей (через СБП или карту РФ)
- Выводят средства пользователям (через СБП или карту РФ)
- Имеют свой баланс, на который могут делать депозиты

## Особенности

- **Множественные методы оплаты**: карты, банковские переводы, криптовалюты, e-wallets
- **KYC/AML проверки**: верификация пользователей и соответствие требованиям
- **Защита от мошенничества**: система мониторинга транзакций
- **Быстрые выплаты**: мгновенная обработка транзакций
- **API для интеграции**: REST API для подключения к основному сайту

## Запуск

```bash
# Установка зависимостей
npm run install:all

# Запуск в режиме разработки
npm run dev
```

- Backend: http://localhost:3002
- Frontend: http://localhost:5179

## API Endpoints

- `GET /api/payment-methods` - Список методов оплаты
- `POST /api/transactions` - Создать транзакцию
- `GET /api/transactions` - Получить транзакции
- `PATCH /api/transactions/:id` - Обновить статус транзакции
- `POST /api/kyc/verify` - KYC верификация
- `GET /api/stats` - Статистика системы
